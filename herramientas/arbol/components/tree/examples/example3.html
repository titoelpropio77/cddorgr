JavaScript Bs_Tree example 3

tree similar to example 1. the special things here are:
  • using the built in checkbox system.
  • the checkbox system can behave in many ways. in this example it works like a windows installer. if you click the box of a node, that information gets inherited down and then up (the tree is walked). open up the folders and click the box of "yahoo sports" to see what i mean. then go down and uncheck "hockey", as you can see "yahoo sports" is now only checked partly.
  • special cases and manual configuration is possible with this checkbox system.
